# Noc POS API

Backend API for Noc POS built with Go and PostgreSQL.

## Features

- User Authentication
- User Management
- Outlet Management
- Product Management
- POS Transactions (ongoing)

## Tech Stack

- Go
- Gin
- PostgreSQL
- Supabase
- Docker

## Run Project

```bash
docker compose up
```

API will run on:

```http
http://localhost:8080
```

## Test API

Use Postman to test available endpoints.

Example:

```http
GET /health

POST /login
```
